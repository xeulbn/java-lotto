로또 게임기능 구현

1. 입력 클래스
- 로또 구입 금액 입력

-> 예외처리 (입력값이 1000원으로 나누어떨어지지 않는 경우)

2. 출력 클래스

1. 로또 발행
- 번호 숫자 범위 1-45
- 1개의 로또당 중복되지 않는 6개의 숫자 뽑기

2. 당첨 번호 추첨
- 중복되지 않는 숫자 6개, 보너스 번호 1개 총 7개의 번호 뽑기

3. 당첨 결과
- 당첨은 1-5등까지
- 1등 (6개번호 일치) / 상금 2,000,000,000원
- 2등 (5개번호 + 보너스번호) / 상금 30,000,000원
- 3등 (5개번호) / 1,500,000원
- 4등 (4개번호) / 50,000원
- 5등 (3개번호) / 5000원
